## 🤖 [VUSiBino Demo](https://github.com/Chafalleiro/VUSiBino-Demo)

Demo host and firmware for VUSBino, an arduino like MCU board that uses V-USB bootloader to load sketches and comunicate with other devices via USB instead of serial port. Can use Arduino IDE or any USB compatible firmware burner. Includes soldering guide and a modified bootloader. [A more complete guide and some other examples.](https://web.archive.org/web/20211129163702/https://chafalladas.com/vusibino-2/). [(Project files and examples)](https://github.com/Chafalleiro/VUSiBino-Demo/releases/tag/v0.1.0)

![GitHub Created At](https://img.shields.io/github/created-at/Chafalleiro/ETRes?link=https://github.com/Chafalleiro/VUSiBino-Demo)


## 🤖 [RTC 4Relay timers](https://github.com/Chafalleiro/RTC-4Relay-timers)

Firmware and schematics to use a battery charged RTC and organize a set of alarms to trigger pinout events. The pinout was connected to a 4 relay board, can be connected to any other similar device. It could trigger more than 8 events, have off hours and holydays programmed. The timers could be set directly via COM console or using a GUI. Was tested in an artistic installation for several months, this are the lastest compilations and sources for the project. [RTC 4Relays GUI.](https://github.com/Chafalleiro/RTC-4Relays-GUI).
 [(RTC files and examples)](https://github.com/Chafalleiro/RTC-4Relay-timers/releases)
 [(GUI files and examples)](https://github.com/Chafalleiro/RTC-4Relays-GUI/releases)

![GitHub Created At](https://img.shields.io/github/created-at/Chafalleiro/ETRes?link=https://github.com/Chafalleiro/RTC-4Relay-timers)
